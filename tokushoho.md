# 特定商取引法に基づく表記

| 項目 | 内容 |
|------|------|
| 販売業者 | 請求があった場合に遅滞なく開示いたします |
| 所在地 | 請求があった場合に遅滞なく開示いたします |
| 電話番号 | 請求があった場合に遅滞なく開示いたします |
| メールアドレス | 211x010x@gmail.com |
| 販売価格 | アプリ内の各サブスクリプションプランに表示される金額 |
| 支払方法 | Apple IDに登録された支払い方法（App Store経由） |
| 支払時期 | 購入手続き完了時に課金 |
| 提供時期 | 購入手続き完了後、即時利用可能 |
| 解約方法 | iPhoneの設定 > [ご自身の名前] > サブスクリプション から管理・解約可能 |
| 返品・キャンセル | デジタルコンテンツの性質上、購入後の返品・返金はAppleの返金ポリシーに準じます |
| 動作環境 | iOS 17.0以降 |
